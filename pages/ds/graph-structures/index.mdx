## When to Use Which

| Need | Structure |
|------|-----------|
| Static path queries | HLD |
| Count paths with property | Centroid Decomposition |
| Sparse key-node queries | Virtual Trees |
| Dynamic link/cut + path queries | Link-Cut Tree |
| Dynamic subtree info | Euler Tour Trees |
