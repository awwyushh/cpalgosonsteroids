## Segment Tree Taxonomy

```
Segment Tree
├── Basic (range query + point update)
│   ├── Recursive
│   └── Iterative (bottom-up)
├── With Lazy Propagation (range update)
│   ├── Range add
│   ├── Range assign
│   └── Multiple operations (compose)
├── Advanced Variants
│   ├── Segment Tree Beats (Ji driver)
│   ├── Merge Sort Tree
│   ├── Segment Tree of Vectors
│   └── Li Chao Tree
├── Persistent Segment Tree
│   └── Persistent Wavelet Tree
├── Dynamic Segment Tree (sparse)
├── 2D Segment Tree
└── Combining with other structures
    ├── HLD + Seg Tree (path queries)
    ├── Seg Tree + Hashing
    └── Seg Tree + BIT (fractal cascading)
```
