## DS Learning Roadmap

```
Level 1 — Beginner
  Array, Prefix Sum, Stack, Queue, Set, Map, BIT

Level 2 — Intermediate
  Segment Tree + Lazy, DSU, Trie, Sparse Table, Mo's Algorithm

Level 3 — Advanced
  Persistent Segtree, Li Chao Tree, Wavelet Tree,
  Rollback DSU, Centroid Decomp, HLD, Implicit Treap

Level 4 — Elite
  Link-Cut Tree, Top Tree, Euler Tour Tree,
  Dynamic Connectivity, Suffix Automaton, Palindromic Tree

Level 5 — Research
  Van Emde Boas, Fusion Trees, Succinct Structures,
  Kinetic Data Structures, Dynamic Wavelet Trees
```
