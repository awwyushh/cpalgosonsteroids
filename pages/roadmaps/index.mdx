# Learning Roadmaps

Choose your path based on your goal and current level.

---

## Paths Overview

| Path | Duration | Goal |
|------|----------|------|
| [Beginner](/roadmaps/beginner) | 4–6 weeks | First contest, Codeforces Div.4/3 |
| [Intermediate](/roadmaps/intermediate) | 3–4 months | Codeforces Expert (1600+) |
| [Advanced](/roadmaps/advanced) | 6–12 months | Codeforces Master (2000+) |
| [Expert](/roadmaps/expert) | 1–2 years | Codeforces Grandmaster / ICPC WF |
| [ICPC](/roadmaps/icpc) | 8–12 months | ICPC Regional / World Finals |
| [Codeforces](/roadmaps/codeforces) | variable | CF rating-focused |
| [IOI / Olympiad](/roadmaps/olympiad) | 1–2 years | IOI Gold/Silver |
| [FFT Mastery](/roadmaps/fft-mastery) | 4–6 weeks | Polynomials & convolutions specialist |
| [Geometry Mastery](/roadmaps/geometry-mastery) | 4–6 weeks | Computational geometry specialist |
| [Game Theory](/roadmaps/game-theory-mastery) | 2–3 weeks | Game theory specialist |

---

## Quick Self-Assessment

**Can you solve Codeforces 1200 problems?** → Start [Beginner](/roadmaps/beginner)

**Can you solve Codeforces 1600 problems?** → Start [Intermediate](/roadmaps/intermediate)

**Can you solve Codeforces 2000 problems?** → Start [Advanced](/roadmaps/advanced)

**Targeting ICPC?** → Follow [ICPC Roadmap](/roadmaps/icpc) regardless of CF rating
