# Beginner Path

**Goal**: Solve Codeforces Div.3/4 problems. Reach 1200–1400 rating.

**Prerequisites**: Basic programming in C++ (variables, loops, arrays, functions).

**Duration**: 4–6 weeks, ~2–3 hours/day.

## Milestone Checkpoints

- [ ] Solve all CSES Introductory Problems
- [ ] Solve 20 Codeforces problems rated 1000–1200
- [ ] Can implement BIT and basic Segment Tree from scratch
- [ ] Can compute $n! \mod p$ and $\binom{n}{k} \mod p$
- [ ] Solved a DSU problem (Kruskal's or connectivity)

---

## First Contest Tips

1. Read **all** problems before starting
2. Start with the easiest problem
3. Test with examples before submitting
4. If stuck for 15 min, move to the next
5. After the contest: read editorials for problems you couldn't solve

---

## What to Practice

- [CSES Problem Set](https://cses.fi/problemset/) — best beginner collection
- Codeforces Div.4 and Div.3 contests
- AtCoder ABC (A, B, C problems)
