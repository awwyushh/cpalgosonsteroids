## Why This Topic Matters

Euler's totient $\phi(n)$ counts integers in $[1, n]$ coprime to $n$. It appears in:
- Euler's theorem: $a^{\phi(n)} \equiv 1 \pmod{n}$ (generalization of Fermat)
- Order of elements in $\mathbb{Z}/n\mathbb{Z}$
- Counting primitive roots, generators
- Möbius inversion and multiplicative function theory

**Contest relevance**: CF Div.1 C–D, number theory problems involving coprimality, Euler's theorem for non-prime moduli.

---

## Learning Objectives

- Compute $\phi(n)$ from prime factorization in $O(\sqrt{n})$
- Build a sieve for $\phi(1), \ldots, \phi(N)$ in $O(N \log \log N)$
- Apply Euler's theorem: $a^{\phi(n)} \equiv 1 \pmod{n}$ for $\gcd(a,n)=1$
- Use the identity $\sum_{d|n} \phi(d) = n$

---

## Definition and Formula

$$\phi(n) = n \prod_{p | n} \left(1 - \frac{1}{p}\right)$$

For $n = p_1^{a_1} \cdots p_k^{a_k}$:

$$\phi(n) = n \cdot \frac{p_1 - 1}{p_1} \cdot \frac{p_2 - 1}{p_2} \cdots \frac{p_k - 1}{p_k}$$

```cpp
long long euler_phi(long long n) {
    long long result = n;
    for (long long p = 2; p * p <= n; p++) {
        if (n % p == 0) {
            while (n % p == 0) n /= p;
            result -= result / p;
        }
    }
    if (n > 1) result -= result / n;
    return result;
}
```

**Examples**: $\phi(1) = 1$, $\phi(p) = p-1$, $\phi(p^k) = p^{k-1}(p-1)$, $\phi(2^k) = 2^{k-1}$.

---

## Multiplicative Property

$\phi$ is multiplicative: $\gcd(m,n)=1 \implies \phi(mn) = \phi(m)\phi(n)$.

Key values:
- $\phi(1) = 1$
- $\phi(p) = p - 1$ for prime $p$
- $\phi(p^k) = p^{k-1}(p-1)$
- $\phi(2) = 1$, $\phi(4) = 2$, $\phi(2^k) = 2^{k-1}$
- $\phi(n) = n/2$ iff $n$ is a power of 2

---

## Phi Sieve

Compute $\phi$ for all $n \leq N$ in $O(N \log \log N)$:

```cpp
const int MAXN = 1e6 + 5;
int phi_arr[MAXN];

void phi_sieve() {
    iota(phi_arr, phi_arr + MAXN, 0);  // phi[i] = i initially
    for (int p = 2; p < MAXN; p++) {
        if (phi_arr[p] == p) {  // p is prime
            for (int n = p; n < MAXN; n += p)
                phi_arr[n] -= phi_arr[n] / p;
        }
    }
}
```

**Linear sieve** — computes $\phi$ in $O(N)$ alongside primes:

```cpp
const int MAXN = 1e6 + 5;
int phi_arr[MAXN], spf[MAXN];
vector<int> primes;

void linear_sieve_phi() {
    iota(phi_arr, phi_arr + MAXN, 0);
    fill(spf, spf + MAXN, 0);
    phi_arr[1] = 1;
    for (int i = 2; i < MAXN; i++) {
        if (!spf[i]) {
            spf[i] = i;
            phi_arr[i] = i - 1;
            primes.push_back(i);
        }
        for (int p : primes) {
            if (p > spf[i] || (long long)i * p >= MAXN) break;
            spf[i * p] = p;
            if (i % p == 0)
                phi_arr[i * p] = phi_arr[i] * p;           // p already in i
            else
                phi_arr[i * p] = phi_arr[i] * (p - 1);    // new prime factor
        }
    }
}
```

**Using SPF table** (per-query, $O(\log n)$ after sieve):
```cpp
int phi_from_spf(int n, int* spf) {
    int result = n;
    while (n > 1) {
        int p = spf[n];
        result -= result / p;
        while (n % p == 0) n /= p;
    }
    return result;
}
```

## Euler's Theorem

For $\gcd(a, n) = 1$:

$$a^{\phi(n)} \equiv 1 \pmod{n}$$

**Corollary** (modular inverse for composite $n$, $\gcd(a,n)=1$):
$$a^{-1} \equiv a^{\phi(n)-1} \pmod{n}$$

---

## Extended Euler's Theorem (Large Exponent)

For $a^b \bmod n$ when $b$ is huge (e.g., tower of powers):

$$a^b \equiv \begin{cases} a^{b \bmod \phi(n)} \pmod{n} & \text{if } \gcd(a,n) = 1 \\ a^{(b \bmod \phi(n)) + \phi(n)} \pmod{n} & \text{if } \gcd(a,n) \neq 1 \text{ and } b \geq \phi(n) \\ a^b \pmod{n} & \text{if } b < \phi(n) \end{cases}$$

The middle case works because for $b \geq \phi(n)$, $a^b$ and $a^{b \bmod \phi(n) + \phi(n)}$ are congruent mod $n$ even when $\gcd(a,n) \neq 1$.

```cpp
// a^b mod n, b can be astronomically large
// Pass b as string if needed; here shown with long long
long long power_mod_euler(long long a, long long b, long long n) {
    if (n == 1) return 0;
    a %= n;
    long long phi_n = euler_phi(n);
    if (__gcd(a, n) == 1) {
        return power(a, b % phi_n, n);
    } else {
        if (b < phi_n) return power(a, b, n);     // small b: compute directly
        return power(a, b % phi_n + phi_n, n);    // large b: extended theorem
    }
}
```

<Callout type="info">
This is needed for "Exponentiation II" (CSES 1712) and any problem with $a^{b^c} \bmod n$ where $b^c$ is too large to compute.
</Callout>

## Key Identities

$$\sum_{d \mid n} \phi(d) = n$$

**Proof**: group integers $1, \ldots, n$ by $\gcd(k, n) = d$. Each group has $\phi(n/d)$ elements.

In Dirichlet convolution notation: $\phi * \mathbf{1} = \text{id}$, or equivalently $\phi = \mu * \text{id}$.

$$\phi(n) = n \sum_{d \mid n} \frac{\mu(d)}{d}$$

---

## Counting Coprimes to n in Range [1, m]

Use **inclusion-exclusion** on the prime factors of $n$:

```cpp
// Count integers in [1, m] coprime to n
long long count_coprime(long long m, long long n) {
    // Get distinct prime factors of n
    vector<long long> primes;
    long long tmp = n;
    for (long long p = 2; p * p <= tmp; p++) {
        if (tmp % p == 0) {
            primes.push_back(p);
            while (tmp % p == 0) tmp /= p;
        }
    }
    if (tmp > 1) primes.push_back(tmp);

    int k = primes.size();
    long long result = 0;
    for (int mask = 0; mask < (1 << k); mask++) {
        long long prod = 1;
        int bits = __builtin_popcount(mask);
        for (int i = 0; i < k; i++)
            if (mask >> i & 1) prod *= primes[i];
        result += (bits % 2 == 0 ? 1 : -1) * (m / prod);
    }
    return result;
}
// count_coprime(n, n) == phi(n)
// count_coprime(m, n) for arbitrary m
```

**Note**: $\phi(n) = n \cdot \prod_{p|n}(1 - 1/p)$ is exactly this inclusion-exclusion applied to $[1,n]$.

---

## Sum of Phi Up to N

$$\Phi(N) = \sum_{k=1}^{N} \phi(k)$$

Using sieve prefix sums: $O(N)$ precomputation, $O(1)$ query.

For large $N$ without precomputation (Mertens function approach):
$$\Phi(N) = \frac{N(N+1)}{2} - \sum_{d=2}^{N} \Phi(\lfloor N/d \rfloor)$$

This gives $O(N^{2/3})$ with memoization.

## Common Patterns

1. **Count coprimes in [1, n]**: $\phi(n)$ directly
2. **Count coprimes to n in [1, m]**: inclusion-exclusion (see above)
3. **Modular inverse for composite n**: $a^{\phi(n)-1} \bmod n$ when $\gcd(a,n)=1$
4. **$a^b \bmod n$ with huge b**: extended Euler theorem
5. **Order of $a \bmod n$**: must divide $\phi(n)$; find smallest such divisor
6. **Number of generators of $\mathbb{Z}/n\mathbb{Z}$**: $\phi(n)$ (when cyclic group exists)
7. **Sum of coprimes to n in [1, n]**: $n \cdot \phi(n) / 2$ for $n > 1$

---

## Common Mistakes

1. **`result -= result / p` order**: do this **before** dividing out all factors of $p$ from $n$. The formula updates multiplicatively on the current result.
2. **Euler's theorem when $\gcd(a,n) \neq 1$**: basic theorem doesn't apply — use extended Euler or CRT.
3. **`phi[1] = 1`**: don't forget to initialize; $\phi(1) = 1$ by convention.
4. **Linear sieve phi**: when $p \mid i$, use `phi[ip] = phi[i] * p`; when $p \nmid i$, use `phi[ip] = phi[i] * (p-1)`. Swapping these is a common bug.
