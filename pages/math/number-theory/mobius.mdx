## Why This Topic Matters

Möbius inversion is the key tool for "summing over divisors" problems. Counting coprime pairs, computing multiplicative functions from their Dirichlet series, sieve-based cancellation — all use $\mu$.

**Contest relevance**: CF Div.1 D/E, IOI-style counting problems, inclusion-exclusion at scale.

---

## Learning Objectives

- Define $\mu(n)$ and compute it via sieve
- State and apply Möbius inversion: $f = g * \mathbf{1} \Leftrightarrow g = f * \mu$
- Count coprime pairs using $\mu$
- Apply to counting squarefree numbers, GCD sums

---

## Definition

$$\mu(n) = \begin{cases} 1 & n = 1 \\ (-1)^k & n = p_1 p_2 \cdots p_k \text{ (distinct primes)} \\ 0 & p^2 \mid n \text{ for some prime } p \end{cases}$$

$\mu$ is multiplicative with $\mu(p) = -1$ and $\mu(p^k) = 0$ for $k \geq 2$.

Key identity: $\sum_{d|n} \mu(d) = [n = 1]$.

## Möbius Sieve

```cpp
const int MAXN = 1e6 + 5;
int mu[MAXN];
bool is_composite[MAXN];
int primes[MAXN], pcnt;

void mobius_sieve() {
    mu[1] = 1;
    for (int i = 2; i < MAXN; i++) {
        if (!is_composite[i]) {
            primes[pcnt++] = i;
            mu[i] = -1;
        }
        for (int j = 0; j < pcnt && (long long)i * primes[j] < MAXN; j++) {
            is_composite[i * primes[j]] = true;
            if (i % primes[j] == 0) {
                mu[i * primes[j]] = 0;
                break;
            }
            mu[i * primes[j]] = -mu[i];
        }
    }
}
```

## Möbius Inversion Formula

**Divisor form**: if $g(n) = \sum_{d|n} f(d)$, then:

$$f(n) = \sum_{d|n} \mu(d) \cdot g(n/d)$$

**Multiple form**: if $g(n) = \sum_{n|d} f(d)$, then:

$$f(n) = \sum_{n|d} \mu(d/n) \cdot g(d)$$

## Counting Coprime Pairs

Count pairs $(a, b)$ with $1 \leq a \leq b \leq n$ and $\gcd(a, b) = 1$:

$$\sum_{k=1}^{n} \mu(k) \lfloor n/k \rfloor^2$$

(Since $[\gcd(a,b) = 1] = \sum_{d | \gcd(a,b)} \mu(d)$.)

```cpp
long long count_coprime_pairs(int n) {
    // Precompute mu via sieve
    mobius_sieve();
    long long result = 0;
    for (int k = 1; k <= n; k++) {
        if (mu[k]) {
            long long q = n / k;
            result += mu[k] * q * q;
        }
    }
    // Pairs (a,b) with a ≤ b: (result + 1) / 2 + adjustment for a=b=1
    return result;
}
```

---

## Count Squarefree Numbers

Count squarefree $n \leq N$: use $\sum_{k=1}^{\lfloor\sqrt{N}\rfloor} \mu(k) \lfloor N/k^2 \rfloor$.

```cpp
long long count_squarefree(long long n) {
    mobius_sieve();
    long long result = 0;
    for (long long k = 1; k * k <= n; k++)
        result += mu[k] * (n / (k * k));
    return result;
}
```

## GCD Sum

$$\sum_{i=1}^{n} \sum_{j=1}^{n} \gcd(i,j) = \sum_{d=1}^{n} d \cdot \phi(d) \cdot \left(2\lfloor n/d \rfloor - 1\right)$$

Using Möbius inversion on $\gcd(i,j) = \sum_{d|\gcd(i,j)} \phi(d)$.

## Common Mistakes

1. **$\mu(p^k) = 0$ for $k \geq 2$**: not $(-1)^k$. Only squarefree numbers have nonzero $\mu$.
2. **Forgetting $[n=1]$ identity**: $\sum_{d|n} \mu(d) = 0$ for $n > 1$, not $-1$.
