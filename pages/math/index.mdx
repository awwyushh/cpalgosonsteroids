## Dependency Graph

```
Foundations
├── Number Theory → Advanced NT
├── Combinatorics → Generating Functions → Polynomials/FFT
├── Linear Algebra → Berlekamp-Massey → Sequences
├── Probability → Markov Chains
├── Geometry (standalone)
├── Game Theory → Algebraic Structures
├── Optimization → Bitmask Math
└── Graph Math ← (Number Theory + Linear Algebra)
```
