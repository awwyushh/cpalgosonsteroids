## 1. Core Graph Theorems

### A. Handshaking Lemma (Degree-Edge Duality)
In any undirected graph $G = (V, E)$, the sum of degrees of all vertices is exactly twice the number of edges:
$$\sum_{v \in V} \text{deg}(v) = 2|E|$$
* **Corollary**: The number of vertices with odd degrees in any graph must be **even**.

### B. Planar Graphs & Euler's Formula
A planar graph is a graph that can be drawn on a 2D plane such that no two edges cross.
For any connected planar graph with $V$ vertices, $E$ edges, and $F$ faces (including the unbounded outer region):

<FormulaBox title="Euler's Planar Formula" formulas={[
  {
    name: "Euler's Relation",
    latex: "V - E + F = 2"
  },
  {

  },

  }
]} />
These bounds are incredibly useful in contests for pruning search branches; if a problem guarantees a planar graph, you know $|E|$ is strictly $O(|V|)$!

## 2. Labeled Trees & Prüfer Sequences
Cayley's formula states that the number of labeled trees on $N$ vertices is exactly **$N^{N-2}$**.
We prove this using a bijection between labeled trees and **Prüfer Sequences** of length $N-2$.

### Prüfer Sequence Construction:
1. Find the leaf (degree 1) with the smallest label.
2. Record its parent's label in the sequence.
3. Remove the leaf from the tree.
4. Repeat until only 2 vertices remain.

```
Example Tree:
    1 - 2 - 3
        |
        4
Leaves: 1, 3, 4. Smallest: 1. Parent: 2. Remove 1. Tree: 2-3, 2-4.
Leaves: 3, 4. Smallest: 3. Parent: 2. Remove 3. Tree: 2-4.
Prüfer Sequence: {2, 2}
```

<Callout type="tip" title="Prüfer Code Properties">
1. A node $v$ appears exactly $\text{deg}(v) - 1$ times in the Prüfer sequence.
2. Nodes that do not appear in the sequence are the initial leaves of the tree.
</Callout>

## 3. Kirchhoff's Matrix-Tree Theorem
To calculate the **number of spanning trees** in any connected graph $G$, we use the Laplacian Matrix.

### The Laplacian Matrix ($L$):
Define $L = D - A$, where $D$ is the diagonal degree matrix and $A$ is the adjacency matrix:
$$L_{ij} = \begin{cases} \text{deg}(i) & \text{if } i = j \\ -1 & \text{if } i \neq j \text{ and } (i, j) \in E \\ 0 & \text{otherwise} \end{cases}$$

### The Theorem:
Select any vertex $r$ (usually the last vertex $N-1$). Form the cofactor submatrix $L^*$ by deleting the $r$-th row and $r$-th column from $L$.
$$\text{Number of Spanning Trees} = \det(L^*)$$

#### C++ Laplacian Spanning Tree Solver:
```cpp
// O(N^3) using Gaussian Elimination modulo a prime
ll power(ll base, ll exp) {
    ll res = 1; base %= MOD;
    while (exp > 0) {
        if (exp & 1) res = (res * base) % MOD;
        base = (base * base) % MOD;
        exp >>= 1;
    }
    return res;
}
ll modInverse(ll n) { return power(n, MOD - 2); }

ll determinant(vector<vector<ll>> A) {
    int n = A.size();
    ll det = 1;
    for (int i = 0; i < n; i++) {
        int pivot = i;
        for (int j = i + 1; j < n; j++) {
            if (abs(A[j][i]) > abs(A[pivot][i])) pivot = j;
        }
        if (A[pivot][i] == 0) return 0;
        if (pivot != i) {
            swap(A[i], A[pivot]);
            det = (MOD - det) % MOD;
        }
        det = (det * A[i][i]) % MOD;
        ll inv = modInverse(A[i][i]);
        for (int j = i + 1; j < n; j++) {
            ll factor = (A[j][i] * inv) % MOD;
            for (int k = i; k < n; k++) {
                A[j][k] = (A[j][k] - factor * A[i][k] % MOD + MOD) % MOD;
            }
        }
    }
    return det;
}

ll count_spanning_trees(int N, const vector<pair<int,int>>& edges) {
    vector<vector<ll>> L(N, vector<ll>(N, 0));
    for (auto& edge : edges) {
        int u = edge.first, v = edge.second;
        L[u][u]++; L[v][v]++;
        L[u][v] = (L[u][v] - 1 + MOD) % MOD;
        L[v][u] = (L[v][u] - 1 + MOD) % MOD;
    }
    // Form cofactor submatrix by deleting the last row/col
    vector<vector<ll>> L_star(N - 1, vector<ll>(N - 1, 0));
    for (int i = 0; i < N - 1; i++) {
        for (int j = 0; j < N - 1; j++) {
            L_star[i][j] = L[i][j];
        }
    }
    return determinant(L_star);
}
```

## 4. Tutte Matrix & Perfect Matchings
For any graph $G$, we define the **Tutte Matrix** $T$ of variables:
$$T_{ij} = \begin{cases} x_{ij} & \text{if } (i, j) \in E \text{ and } i < j \\ -x_{ij} & \text{if } (i, j) \in E \text{ and } i > j \\ 0 & \text{otherwise} \end{cases}$$

### Tutte's Theorem:
The symbolic determinant of the Tutte matrix $T$ is non-zero **if and only if** the graph contains a perfect matching.
* **Randomized Tutte Test (Schwartz-Zippel Lemma)**: Substitute random values from a large finite field $\mathbb{F}_p$ for the variables $x_{ij}$. If the determinant is non-zero, the graph has a perfect matching with probability $> 1 - \frac{V}{p}$.

## 5. Electrical Networks & Random Walks
Let's represent a graph as a circuit where each edge is a $1\ \Omega$ resistor.
* **Voltages & Currents**: If we connect a $1\text{V}$ battery between source $s$ and sink $t$, the node voltages satisfy Kirchhoff's Current Law:
  $$V_u = \frac{1}{\text{deg}(u)} \sum_{v \sim u} V_v \quad (u \neq s, t)$$

This is identical to a **Random Walk**:
* The voltage $V_u$ is exactly the probability that a random walk starting at $u$ reaches the target $s$ before reaching $t$!
* **Effective Resistance ($R_{\text{eff}}$)**: The effective resistance between $s$ and $t$ is:
  $$R_{\text{eff}}(s, t) = \frac{1}{I_{\text{total}}}$$
  This is directly related to the **Commute Time** (expected steps for a random walk to go $s \to t \to s$):
  $$\text{CommuteTime}(s, t) = 2|E| \cdot R_{\text{eff}}(s, t)$$
